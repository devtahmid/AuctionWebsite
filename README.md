# AuctionWebsite
An auction website created as part of a school project. It was tested on Apache server and utilises MySql database. 
Demo website: vowageboat.study

### Website functionalites:  
1. login/sign up
2. Create auction
3. Browse Auctions
4. Bid on auction
5. Notifications 
6. Messages 
7. Profile
